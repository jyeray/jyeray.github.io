My Gsoc Project With Opensuse

  • Joaquín Martín

    @jyeraydev

Hello openSUSE community, my name is Joaquín (my nickname is jyeray), I’m from Canary Islands, Spain. And here you can see my Github. I am studying Software Engineering at University of Las Palmas de Gran Canaria. I’m an enthusiast of free software and GNU/Linux, I have tasted many GNU/Linux distributions but when I install openSUSE … I love it!.

I’m excited to be able to contribute to openSUSE in the Google Summer of Code because I have the opportunity to take advantage of my mentors, Josef Reidinger and Ancor Gonzalez Sosa, two proficient YaST developers. I have many things to learn from them.

I think that Google Summer of Code is the best opportunity to get introduced to open source development. Personally I chose openSUSE because it has a great community, as I said before i love openSUSE distribution, and I will have the opportunity to work on one of the best tools of openSUSE, YaST, this powerful tool that makes it all easier. Also working with one of the languages that had pending to learn, ruby.

My project is Alternative YaST Module, the idea is to develop a YaST module to manage update-alternatives with an easy and simple GUI. In this module you will be able to see the all alternatives group, manually change an alternative, see all the alternatives of an alternatives group and his slaves and set an alternative group in automatic mode. You can view the project status on Github

Finally I want to thank my mentors and openSUSE for giving me this great opportunity. I really want to work on this project, I think this is going to be an amazing summer!
