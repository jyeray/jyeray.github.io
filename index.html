MiddlewareFilter asp .net core

One of my favourites features of ASP .NET Core compared to ASP .NET is the application pipeline or app pipeline. The app pipeline allows you to specify what should be executed when an HTTP request arrives.

The idea is simple when an HTTP request is executing, the middlewares are executed in order. Each middleware has the responsibility to call the next middleware in the pipeline or do not call it, short-circuiting the pipeline. This is a powerful way to configure the behaviour of your application.
